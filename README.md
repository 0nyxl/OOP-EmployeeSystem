# OOP-EmployeeSystem (C++)

A console-based Employee Management System built in **C++**, demonstrating **object-oriented programming (OOP)** concepts.  
This project focuses on **encapsulation, static members, and structured console output**.

---

## Features

- **Employee class** with private member variables  
- **Automatic Employee ID** assignment using a static member  
- **Getters and Setters** for all employee fields  
- **Formatted console output** with alignment for readability  
- Clean, maintainable C++ code structure  

---

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/0nyxl/OOP-EmployeeSystem.git
   cd OOP-EmployeeSystem
   ```
2.Compile the program:
```bash
g++ main.cpp -o EmployeeApp
```
2.Run the program:
```bash
./EmployeeApp   # Linux / macOS
EmployeeApp.exe # Windows
